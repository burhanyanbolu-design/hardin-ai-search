STEP 1: CREATE GITHUB REPOSITORY
─────────────────────────────────

1. Go to: https://github.com/new

2. Fill in:
   Repository name: hardin-ai-search
   Description: AI tool search engine powered by self-replicating bot army
   ✅ Public
   ❌ Don't check "Add README"

3. Click: "Create repository"

4. COPY the repository URL:
   https://github.com/YOUR_USERNAME/hardin-ai-search.git


STEP 2: PUSH CODE TO GITHUB
────────────────────────────

Open Command Prompt in ai-search-engine folder:

cd Desktop\SmartTracker-AI\ai-search-engine

Then run these commands:

git init
git add .
git commit -m "Initial commit - Hardin AI Search Engine"
git branch -M main
git remote add origin https://github.com/YOUR_USERNAME/hardin-ai-search.git
git push -u origin main

(Replace YOUR_USERNAME with your GitHub username)


STEP 3: DEPLOY TO VERCEL
─────────────────────────

1. Go to: https://vercel.com

2. Click: "Sign Up" or "Login"
   → Use "Continue with GitHub"

3. Click: "Add New" → "Project"

4. Find your repository: hardin-ai-search
   Click: "Import"

5. Configure:
   Framework Preset: Other
   Root Directory: ./
   Build Command: (leave empty)
   Output Directory: (leave empty)

6. Click: "Deploy"

7. Wait 1-2 minutes...

8. ✅ DONE! You'll get a URL like:
   https://hardin-ai-search.vercel.app


STEP 4: TEST YOUR DEPLOYMENT
─────────────────────────────

TROUBLESHOOTING:

❌ Git not found?
   → Install from: https://git-scm.com/download/win

❌ Push failed?
   → Check GitHub username/password
   → Use Personal Access Token if needed

❌ Vercel deployment failed?
   → Check vercel.json exists
   → Check requirements.txt is correct
   → View logs in Vercel dashboard

❌ Search not working?
   → Check data/tools folder has JSON files
   → Verify API endpoints work
   → Check browser console for errors
